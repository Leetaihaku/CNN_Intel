Intel Dataset을 기반으로한 CNN 다중 분류기(Multi-Classifier)
각 레이블별 훈련데이터 개수는 약 2300여개 남짓
분류기 평균 성능 96%

TIP
 - 전결합층의 노드 수는 Global Optima Convergency에 영향 적음
 - 전결합층의 층계 수는 영향 있음(양에 따라 고차원 판단, But 보편적 2계층 적절 => 수렴 한계오면 마지막 변수로서 증감시켜볼 것!)
 - ConvNet은 점진적 형태로 층계구성할 것! 추가적으로 층계 간 필터개수를 등차수열 형식으로, 절대 비율형식으로 크게 늘리지말 것!
 - ConvNet 역시, 보편적인 경우 2계층 적절
 - ConvNet 초기 설정 시, 선행 계층 노드 수와 동일하게 이후 계층 통일 후 하나씩 노드 최적 수 검색
 - Image Preprocessing 필수 -> [-1, 1] 사이 정규화시 fitting에 상당한 이점!
 - Dropout과 L1, L2와 같은 정규화는 Overfitting 발생 전까지 우선적으로 고려하지말 것!
 - 학습률이 일정 이상 오르지 않을 시, 경사도 발산을 의심해볼것!
